Summary

Exposes methods to GET and SET Library Descriptor information as per define in DatabaseDescriptor.si.xml or LibraryDescriptor.si.xml file by application.

Example: java_variable_data_type true/false true/false true/false condition_to_be_checked (Eg: variable_name 'condition' value; variable_name > 0) default_value_of_column (Eg: 0.1) column_name_needs_to_add true/false true/false true/false true/false

BeginTransaction(DatabaseDescriptor databaseDescriptor)

Begins a transaction in EXCLUSIVE mode.

Transactions can be nested. When the outer transaction is ended all of the work done in that transaction and all of the nested transactions will be committed or rolled back. The changes will be rolled back if any transaction is ended without being marked as clean(by calling commitTransaction). Otherwise they will be committed. Book cBook = new Book(); cBook.SetTitle(Book.BOOK_TYPE_C); cBook.SetDescription("c_description"); cBook.SetAuthor("c_author"); cBook.SetLink("c_link"); DatabaseDescriptor databaseDescriptor = cBook.GetDatabaseDescriptor(); try { Database.BeginTransaction(databaseDescriptor); cBook.Save(); Database.CommitTransaction(databaseDescriptor); } catch(DatabaseException de) { //Log it. } finally { Database.EndTransaction(databaseDescriptor); }

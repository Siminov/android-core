Start()

It is the entry point to the SIMINOV FRAMEWORK.

When application starts it should call this method to activate SIMINOV-FRAMEWORK, by providing ApplicationContext as the parameter. Siminov will read all descriptor defined by application, and do necessary processing. EXAMPLE There are two ways to make a call. Call it from Application class. public App() { this.InitializeComponent(); this.Suspending += this.OnSuspending; new Siminov.Core.Sample.Siminov().InitializeSiminov(); }

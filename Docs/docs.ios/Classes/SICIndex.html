Overview

Exposes methods to GET and SET Reference Map information as per define in DatabaseMappingDescriptor.si.xml file by application.

Example:

<index name="LIQUOR_INDEX_BASED_ON_LINK" unique="true">
    <column>HISTORY</column>
</index>
